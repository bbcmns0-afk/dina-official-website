# Simple Website

This project is a basic static website built using **HTML** and
**CSS**.\
It includes three main pages and a stylesheet for styling.

## Project Structure

    .
    ├── index.html       # Home page
    ├── about.html       # About page
    ├── contact.html     # Contact page
    ├── style.css        # Main stylesheet
    └── favicon.ico      # Website icon

## Features

-   Clean and simple layout\
-   Easy to customize\
-   Includes favicon\
-   Lightweight and fast

## How to Use

1.  Download or extract the project files.
2.  Open **index.html** in any web browser.
3.  Edit the HTML or CSS files as needed.

## Pages

### index.html

Main landing page.

### about.html

Information about the website or project.

### contact.html

Contact information or form layout.

## Styling

All design elements are handled in **style.css**.

## Favicon

The **favicon.ico** file provides the small icon shown in the browser
tab.
